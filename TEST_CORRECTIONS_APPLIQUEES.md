## ?? SI UN TEST éCHOUE

### Probléme : Bouton toggle toujours trop bas

**Solution** :
1. Vérifiez que vous avez bien recompilé (F6)
2. Relancez l'application (F5)
3. Si toujours probléme, vérifiez `Form1.Designer.cs` ligne ~380

---

### Probléme : Onglet Minéraux toujours vide

**Solution** :
1. Fermez Visual Studio complétement
2. Supprimez `bin/` et `obj/`
   ```powershell
   Remove-Item -Recurse -Force bin, obj
   ```
3. Rouvrez Visual Studio
4. Recompilez : Ctrl+Shift+B
5. Relancez : F5

---

### Probléme : Textes toujours invisibles

**Vérifiez** :
1. Le fichier `Forms/MineralsPanel.cs` a bien été modifié
2. La ligne contient `TableLayoutPanel` au lieu de `Panel`
3. Les labels ont `BackColor = Color.Transparent`
