## ?? RECHERCHE PAR ZONE

### Comment trouver les minéraux d'une zone ?

Le service `MineralDataService` propose des méthodes de recherche :

```csharp
// Recherche par localité
var minerals = MineralDataService.SearchByLocality("Collobriéres");
// Retourne : Cuivre, Grenats, Tourmaline, Andalousite, etc.

// Recherche par région
var mineralsMaures = MineralDataService.GetMineralsByRegion("maures");
var mineralsEsterel = MineralDataService.GetMineralsByRegion("esterel");
```

### Sécurité

?? **DANGER** : Les anciennes mines sont DANGEREUSES
- Risque d'effondrement
- Gaz toxiques possibles
- Puits profonds
- Galeries instables
