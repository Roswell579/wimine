## ?? STYLE DE BOUTONS PROPOSé

### Bouton Standard
```csharp
var btn = new Button
{
    FlatStyle = FlatStyle.Flat,
    BackColor = Color.FromArgb(0, 150, 136),  // Vert Teal
    ForeColor = Color.White,
    Font = new Font("Segoe UI", 11, FontStyle.Bold),
    Height = 40,
    FlatAppearance = 
    {
        BorderSize = 0,
        MouseOverBackColor = Color.FromArgb(0, 172, 158),  // Teal plus clair
        MouseDownBackColor = Color.FromArgb(0, 121, 109)   // Teal plus foncé
    }
};

// Ombre portée (simulation)
btn.Paint += (s, e) =>
{
    using (var path = new GraphicsPath())
    {
        path.AddRectangle(btn.ClientRectangle);
        using (var brush = new SolidBrush(Color.FromArgb(50, 0, 0, 0)))
        {
            e.Graphics.TranslateTransform(0, 2);
            e.Graphics.FillPath(brush, path);
            e.Graphics.ResetTransform();
        }
    }
};
```
